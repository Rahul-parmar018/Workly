# Workly

Workly is a modern Android application designed for service management, featuring professional UI and seamless data synchronization.

## 🚀 Launch Steps

To get the project up and running on your local machine, follow these steps:

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/Rahul-parmar018/Workly.git
    ```
2.  **Open in Android Studio**
    Launch Android Studio and select **Open**, then navigate to the cloned `Workly` folder.
3.  **Configure Firebase**
    Ensure that a valid `google-services.json` file is present in the `app/` directory. If you are setting up your own Firebase project, download the file from the Firebase Console.
4.  **Sync Gradle**
    Android Studio should automatically prompt you to sync Gradle files. If not, go to **File > Sync Project with Gradle Files**.
5.  **Build and Run**
    Select your emulator or physical device and click the **Run** (green play) button.
